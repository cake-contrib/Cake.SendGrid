# Cake.SendGrid

[![License](http://img.shields.io/:license-mit-blue.svg)](http://cake-contrib.mit-license.org)

Cake.SendGrid is an Addin for [Cake](http://cakebuild.net/) which allows sending of email via SendGrid.
